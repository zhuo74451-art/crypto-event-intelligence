# Canonical Corpus V1

## Purpose

The Canonical Corpus is the foundation source record library for the Research Intelligence system. It provides a curated, auditable set of research materials across all 13 market cognition domains.

## Selection Criteria

Each source record was selected to meet:
1. **Domain coverage** — All 13 domains represented, with P0 domains prioritized
2. **Role diversity** — Foundational mechanism, empirical update, counterevidence, methodology, data quality
3. **Access diversity** — Open access, public web, and paywalled (metadata-only)
4. **Temporal range** — Classic papers (2000s) through recent (2024)
