# Validation System V1

## Architecture
The Lane D validation system provides point-in-time validation dataset construction,
chronological splitting with purge/embargo, walk-forward analysis, B1-B10 baselines,
bootstrap uncertainty estimation, multiple-testing control, probability calibration,
abstention analysis, drift detection, and leakage auditing.
