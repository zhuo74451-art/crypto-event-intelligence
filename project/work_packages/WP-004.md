- Entrypoint reads producer locks rather than embedded fixtures.

## Cheapest falsification

- Strongest counterexample: The default command succeeds after producer artifacts are removed because it falls back to samples.
- Cheapest test: Remove one producer from a temporary copy and require a stable non-zero failure.

## Stop conditions

- Real producer integration requires a new external service.
- The pipeline cannot preserve point-in-time lineage.
- The integration would emit unsupported inference.
- A safe failure state cannot be represented.
